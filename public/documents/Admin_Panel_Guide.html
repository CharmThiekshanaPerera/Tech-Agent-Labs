1. Login & Authentication

Access the admin panel at /admin-login. Only users with the admin role can sign in. The system uses email/password authentication with role-based access control (RBAC).

Key Features

  • Secure login — Email + password authentication via the backend auth system.
  • Role verification — After login, the system checks the user_roles table to confirm admin status.
  • Password reset — Admins can request a password reset email from the login page.
  • Session persistence — Sessions are maintained across browser refreshes.
  • Activity logging — Every admin login is recorded in the activity log.

2. Dashboard Overview

The main dashboard (/admin) provides a bird's-eye view of your website's key metrics and recent activity.

What You'll See

  • KPI Cards — Total blog posts, contact messages (with unread count), demo bookings (with pending count), testimonials, and total site visits.
  • Site Visits Chart — Area chart showing daily visitor trends over the last 30 days.
  • Messages Over Time — Area chart of incoming contact messages.
  • Demo Bookings Chart — Bar chart of demo booking submissions over time.
  • Demo Status Pie Chart — Visual breakdown of demo statuses (pending, confirmed, completed, cancelled).
  • Recent Activity Log — The latest 15 admin actions (e.g., logins, blog edits, message reads).

3. Analytics

The Analytics page (/admin/analytics) offers deeper insights into lead generation and conversion performance.

Features

  • Time Range Selector — Toggle between 7-day, 30-day, and 90-day views.
  • KPI Cards — Total leads, demo bookings, contact messages, and period-over-period change percentage.
  • Conversion Funnel — Visual funnel from site visitors → contact submissions → demo bookings, with conversion rates.
  • Trends Tab — Area and bar charts showing contact messages and demo bookings over time.
  • Demo Status Tab — Detailed breakdown of demo statuses with individual cards and pie chart.
  • Lead Sources Tab — Top inquiry types from contact form subjects, ranked with progress bars.
  • Response Metrics — Pie chart showing replied vs. unread vs. read message statuses.

4. Revenue Tracking

The Revenue page (/admin/revenue) is a manual revenue dashboard for tracking sales and financial performance.

Features

  • Add Revenue Entry — Record sales, subscriptions, or one-time payments with details like agent name, customer info, amount, date, status, and notes.
  • Edit & Delete — Modify or remove existing entries.
  • KPI Summary — Total revenue, Monthly Recurring Revenue (MRR), total number of sales, and growth percentage.
  • Revenue Trend Chart — Area chart showing revenue over time.
  • Revenue by Agent — Bar chart breaking down revenue by AI agent product.
  • Recurring Payments — Mark entries as recurring with configurable intervals (monthly, quarterly, yearly).
  • CSV Export — Download all revenue data as a CSV file for external analysis.

5. AI Chat Logs

The AI Chat Logs page (/admin/chat-logs) lets you review every conversation visitors have had with the website's AI chatbot.

Features

  • Conversation List — Browse all chat sessions, sorted by most recent. Shows first message preview, message count, and timestamp.
  • Search — Search conversations by first message content or session ID.
  • Conversation Detail — Click any conversation to see the full chat history with user/bot message bubbles.
  • Metadata — View the page path where the chat originated and the exact timestamp.
  • Stats Cards — Total conversations, total messages exchanged, and today's chat count.
  • Delete — Remove individual chat logs.

6. Blog Posts

The Blog Posts page (/admin/blog) provides full content management for the website's blog.

Features

  • Create Posts — Write new blog posts with title, excerpt, content (Markdown supported), category, and header image URL.
  • Edit & Delete — Modify existing posts or remove them entirely.
  • Publish/Draft Toggle — Control whether a post is publicly visible or saved as a draft.
  • Search & Filter — Search posts by title, excerpt, or category.
  • AI Blog Generation AI Powered — Automatically generate complete blog posts using AI, with options to:
    • Select a topic from predefined categories (AI Agents, Automation, Customer Service, etc.)
    • Choose a blog category
    • Auto-publish immediately or save as draft
    • Auto-generate a header image
  • Preview — View the rendered Markdown content with syntax highlighting before publishing.
  • Notifications — When a post is published, newsletter subscribers are automatically notified via email, and the post is shared to configured social channels.

10. Demo Bookings

The Demo Bookings page (/admin/demos) manages all demo requests submitted by potential customers.

Features

  • Booking List — View all demo requests with contact name, company, preferred date/time, status, and submission date.
  • Status Filter — Filter bookings by status: All, Pending, Confirmed, Completed, or Cancelled.

11. Contact Messages

The Messages page (/admin/messages) displays all contact form submissions from the website.

Features

  • Inbox View — Table showing sender name, email, subject, replied status, and date.
  • Unread Indicators — Unread messages are highlighted with a blue dot and bold text. The header shows the unread count.
  • Auto-Mark as Read — Messages are automatically marked as read when you view the detail.
  • Reply Tracking — Toggle a "replied" status for each message to track which ones you've responded to.
  • View Details — Open a detail dialog with full sender info, message content, and timestamp.
  • Reply via Email — One-click button to open your email client with the sender's address pre-filled.
  • Delete — Remove messages with confirmation.

13. SEO & Performance

The SEO & Performance page (/admin/seo) provides tools for monitoring and optimizing the website's search engine visibility and load speed.

Features

  • SEO Checklist — Automated scan of all pages checking for:
    • Page titles and meta descriptions
    • Open Graph tags (og:title, og:description, og:image)
    • JSON-LD structured data
    • Canonical tags
    • Robots.txt and sitemap.xml
  • Page SEO Status — Per-page breakdown showing title, description, OG presence, schema presence, and OG image for every route.
  • Lighthouse Audit Runner Live Audit — Run Google PageSpeed Insights audits for both mobile and desktop, showing:
    • Performance, Accessibility, Best Practices, and SEO scores
    • Core Web Vitals (LCP, FID, CLS, FCP, TTFB)
    • Specific opportunities and diagnostics with savings estimates
  • Export Results — Export full audit results as a formatted text report.
  • External Tools — Quick links to Google Search Console, PageSpeed Insights, Rich Results Test, Schema Validator, Mobile-Friendly Test, and Security Headers checker.

14. Settings

The Settings page (/admin/settings) provides account management options for the logged-in admin.

Features

  • Change Password — Update your admin account password with new + confirm password fields and visibility toggles.
  • Validation — Ensures passwords match and meet minimum length requirements before submission.

General Notes

  • Activity Logging — All significant admin actions (logins, CRUD operations, status changes) are automatically logged and visible on the dashboard.
  • Responsive Design — The admin panel is fully responsive with a collapsible sidebar on mobile devices.
  • Role-Based Access — Only users with the "admin" role can access any admin page. Unauthorized users are redirected to the login page.
  • Real-Time Data — Most pages fetch live data from the database on load with loading indicators.
  • Toast Notifications — Success and error messages appear as toast notifications for all actions.
